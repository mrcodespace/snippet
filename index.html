Markdown-Formatierung Guide

Willkommen zu deinem Markdown-Formatierung Guide. Hier erfährst du, wie du Text mithilfe von Markdown einfach und effektiv formatieren kannst. Nutze die nachstehenden Anweisungen, um deine Texte zu strukturieren und hervorzuheben.

1. Überschrift 1

Erstelle eine große Überschrift, indem du ein # vor deinen Text setzt.

# Text

Text

2. Überschrift 2

Für eine mittlere Überschrift, füge ## vor dem Text hinzu.

## Text

Text

3. Überschrift 3

Eine kleinere Überschrift kannst du mit ### erzeugen.

### Text

Text

4. Fett

Um Text fett zu formatieren, setze ihn zwischen zwei Doppelpunkte **.

**Text**

Text

5. Kursiv

Für kursiven Text verwende ein Sternchen * oder zwei Unterstriche _ um den Text herum.

*Text* oder _Text_

Text

6. Unterstrichen

Text unterstreichen kannst du mit zwei Doppelt-Unterstrichen __.

__Text__

Text

7. Durchgestrichen

Setze zwei Tilden ~~ vor und nach dem Text, um ihn durchzustreichen.

~~Text~~

Text

8. Codeblock (Einzeilig)

Einen Text als Code anzeigen lassen, indem du ihn zwischen zwei Backticks ` setzt.

`Text`

Text

9. Codeblock (Mehrzeilig)

Für mehrzeiligen Code nutze drei Backticks ``` am Anfang und Ende.

```
Text
```
Text

10. Blockzitat

Mit > am Anfang der Zeile erstellst du ein Blockzitat.

> Text
Text

11. Spoiler

Verstecke Text als Spoiler, indem du ihn zwischen zwei vertikale Balken || setzt.

||Text||

Text

12. Kombinationen

Du kannst mehrere Effekte miteinander kombinieren, um komplexe Formatierungen zu erstellen.

***__Text__***

Text
